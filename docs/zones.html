Les données mobilisées

Les deux types de données géographiques

Nous avons mobilisé deux types de données géographiques pour répondre à notre problématique : les vecteurs et les rasters.

Récapitulatif des données

Donnée Type Format Librairie R Rôle dans le projet
MNT (RGE Alti) Raster .tif / SpatRaster terra Modéliser le relief et les zones d’ombre.
Contours Communaux Vecteur .gpkg / sf sf Délimiter la zone d’étude (Roscoff/Plougasnou).
Carroyage Filosofi Vecteur .shp / .gpkg sf Analyser la population à fine échelle (200m).
Données INSEE Table .csv / .xlsx base / dplyr Contenir les variables (âge, résidences sec.).
Monuments Hist. Vecteur .gpkg sf Identifier les points de vue sensibles (clochers).

A. Téléchargement des limites administratives

point <- st_sfc(st_point(c(-3.85, 48.65)), crs = 4326)
zone <- st_buffer(st_transform(point, 2154), 15000)

communes <- get_wfs(x = zone,
                    layer = "ADMINEXPRESS-COG-CARTO.LATEST:commune")

zone_etude <- communes[communes$nom_officiel %in% c("Roscoff", "Plougasnou"), ]

# Colorier explicitement chaque commune
couleurs <- ifelse(zone_etude$nom_officiel == "Plougasnou", "#2196F3", "#FF9800")

plot(st_geometry(zone_etude), 
     col = couleurs,
     main = "Zone d'étude : Roscoff et Plougasnou")
legend("topright", legend = c("Plougasnou", "Roscoff"),
       fill = c("#2196F3", "#FF9800"))

B. Téléchargement du carreau Filosofi (INSEE)

grille <- st_read("C:/Users/damie/Documents/carreaux_200m_met.gpkg", quiet = TRUE)

grille_proj <- st_transform(grille, 2154)
zone_etude_proj <- st_transform(zone_etude, 2154)

idx <- st_intersects(grille_proj, st_union(zone_etude_proj), sparse = FALSE)[,1]
grille_zone <- grille_proj[idx, ]

couleurs_proj <- ifelse(zone_etude_proj$nom_officiel == "Plougasnou", "#2196F3", "#FF9800")

plot(st_geometry(zone_etude_proj), 
     col = couleurs_proj,
     main = "Carreau Filosofi 200m — Roscoff et Plougasnou")
plot(st_geometry(grille_zone), 
     col = "#eef2ff88", border = "#818cf8", 
     add = TRUE)
plot(st_geometry(zone_etude_proj), 
     col = NA, border = "black", lwd = 2,
     add = TRUE)
legend("topright", legend = c("Plougasnou", "Roscoff"),
       fill = c("#2196F3", "#FF9800"))

C. Jointure spatiale sur les communes

grille_communes <- st_join(grille_zone, zone_etude_proj["nom_officiel"])

plot(st_geometry(zone_etude_proj), 
     col = NA, border = "black", lwd = 2,
     main = "Carreau Filosofi découpé par commune")
plot(st_geometry(grille_communes[!is.na(grille_communes$nom_officiel) & 
                                  grille_communes$nom_officiel == "Plougasnou", ]),
     col = "#bfdbfe", border = "#3b82f6", add = TRUE)
plot(st_geometry(grille_communes[!is.na(grille_communes$nom_officiel) & 
                                  grille_communes$nom_officiel == "Roscoff", ]),
     col = "#fed7aa", border = "#f97316", add = TRUE)
legend("topright", 
       legend = c("Plougasnou", "Roscoff"),
       fill = c("#bfdbfe", "#fed7aa"))
